**AI 기반 민법 조문 및 판례 O/X 학습 플랫폼** 🔗 **[Law Quizzer 바로가기](https://kgw0357.github.io/lawquizzer/)**

---

## 📖 프로젝트 소개
**Law Quizzer**는 법학도 및 수험생을 위한 스마트 AI 법학 학습 도우미입니다. 공부하고 있는 민법 조문이나 판례 문단을 입력하면, **Google Gemini AI**가 텍스트를 즉각적으로 분석하여 핵심적인 O/X 문제를 출제합니다. 

단순한 문제 풀이를 넘어 즉각적인 해설과 오답 노트 기능을 제공하여, 사용자가 법리적 사고를 효과적으로 연마하고 취약점을 보완할 수 있도록 돕습니다.

<br>

## ✨ 주요 기능

### 1. 🤖 AI 자동 출제 시스템 (Powered by Gemini AI)
- 학습할 법령 및 판례 텍스트(최대 4,000자)를 입력하면 AI가 핵심 논점 5가지를 추출하여 O/X 퀴즈로 변환합니다.
- 처음 이용하는 사용자를 위해 **빠른 예시 테스트**(하자담보책임, 증여 계약, 임대차 관련 등) 기능을 제공합니다.

### 2. ⚡ 실시간 해설 및 채점
- 참(O) / 거짓(X) 선택 시 즉각적인 정답 여부를 피드백합니다.
- 채점과 동시에 관련 법적 근거(민법 조항 등)가 포함된 **상세한 해설**을 제공하여 깊이 있는 학습을 유도합니다.

### 3. 📚 스마트 오답 노트
- 퀴즈 중 틀린 문제는 사용자의 **오답 노트**에 자동으로 저장됩니다.
- 나의 취약점 리스트를 언제든 복습할 수 있으며, 완벽히 이해한 문제는 '이해함' 버튼을 눌러 리스트에서 삭제할 수 있습니다.

### 4. 🔐 개인 맞춤형 학습 관리
- 이메일 기반의 간편한 회원가입 및 로그인을 지원합니다.
- 계정별로 오답 노트와 학습 상태(누적 오답 수 등)를 안전하게 기록 및 관리합니다.

### 5. ⚙️ 사용자 맞춤형 API 설정
- 원활하고 독립적인 AI 사용을 위해 사용자의 개인 **Gemini API Key**를 직접 입력하여 구동할 수 있습니다.
- *보안:* 입력된 API 키는 서버로 전송되지 않고 브라우저의 로컬 저장소(Local Storage)에만 안전하게 보관됩니다.

<br>

## 💻 사용 방법

1. **로그인 / 회원가입**: [웹사이트](https://kgw0357.github.io/lawquizzer/) 접속 후 이메일 계정으로 로그인합니다.
2. **학습 자료 입력**: 메인 화면의 텍스트 에디터에 학습하고 싶은 민법 조문이나 판례 문단을 붙여넣습니다.
3. **퀴즈 생성**: `🔍 퀴즈 분석 및 자동 생성` 버튼을 클릭합니다.
4. **퀴즈 풀이**: AI가 생성한 5개의 O/X 퀴즈를 풀고, 실시간으로 출력되는 해설을 통해 법리를 파악합니다.
5. **오답 복습**: 학습 종료 후 상단의 `📚 오답 노트` 메뉴로 이동하여 틀렸던 문제를 다시 복습합니다.

프로젝트를 로컬 환경에서 실행하거나 수정하시려면 아래의 단계를 따라주세요.

# 3. 웹 서버 실행 (Launch the web server)
# 별도의 빌드 과정이 없는 Vanilla 환경이라면, Live Server 익스텐션 등을 사용해 index.html을 실행하세요.
# Node.js 패키지 매니저를 사용하는 경우:

npm start
